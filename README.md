__Учебный проект по автоматизации тестирования поиска в Гугл__

Для установки всех зависимостей можно воспользоваться командой:

      pip install -r requirements.txt

Для корректной работы тестов может потребоваться VPN. 
Также может помочь отключение картинок с помощью кастомного флага 
--images=false для Pytest:

      pytest --images=false

Тесты в фоновом режиме (headless), а также с помощью браузера Firefox, 
не проходят из-за капчи Гугл. Работает только Chrome в обычном режиме.


<br>__Задание__

Автоматизировать проверку двух сценариев.


<br>__Сценарий 1: Поиск в Гугл__

1. Зайти на google.com

2. Проверить наличие поля поиска

3. Ввести в поиск 'Яндекс'

4. Проверить, что появился список с подсказками

5. Проверить, что в списке подсказок есть слово 'яндекс' (не в рекламном блоке)

6. Выполнить поиск по ключевому слову 'Яндекс'. Выполнить поиск возможно двумя способами:
          
    * Нажать ENTER

    * Нажать кнопку 'Поиск в Google'

7. Поверить:

    * Появляется таблица результатов поиска
   
    * В первых 5 результатах поиска есть ссылка на yandex.ru


<br>__Сценарий 2: Картинки в Гугл__

1. Выполнить все действия из Сценария 1

2. Проверить, что кнопка 'Картинки' присутствует на странице

3. Нажать на кнопку 'Картинки'

4. Открыть вторую картинку, проверить, что она открылась

5. При нажатии кнопки 'Следующее изображение' картинка должна измениться

6. Нажать на кнопку 'Предыдущее изображение'

7. Проверить, что картинка изменилась на картинку из шага 4
